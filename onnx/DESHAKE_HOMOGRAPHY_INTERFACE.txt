================================================================================
HOMOGRAPHY MATRIX FORMAT (OPENCV)
================================================================================

The homography matrix is a 3x3 matrix that represents 2D transformations:

    H = [h00 h01 h02]
        [h10 h11 h12]
        [h20 h21 h22]

Where:
- h00, h01, h10, h11: Rotation and scaling
- h02, h12: Translation (tx, ty)
- h20, h21: Perspective (shear)
- h22: Scale (usually 1)

Transformation:
    [x', y', w']^T = H * [x, y, 1]^T
    
    x' = h00 * x + h01 * y + h02
    y' = h10 * x + h11 * y + h12
    w' = h20 * x + h21 * y + h22
    
    x_norm = x' / w'
    y_norm = y' / w'

================================================================================
COMMON HOMOGRAPHY MATRICES
================================================================================

1. Translation Only:
   H = [1  0  tx]
       [0  1  ty]
       [0  0   1]

2. Rotation (angle θ):
   H = [cos(θ)  -sin(θ)  0]
       [sin(θ)   cos(θ)  0]
       [0        0       1]

3. Scaling (sx, sy):
   H = [sx  0   0]
       [0   sy  0]
       [0   0   1]

4. Rotation + Translation:
   H = [cos(θ)  -sin(θ)  tx]
       [sin(θ)   cos(θ)  ty]
       [0        0       1]

5. Full Affine (rotation + scaling + translation):
   H = [sx*cos(θ)  -sy*sin(θ)  tx]
       [sx*sin(θ)   sy*cos(θ)  ty]
       [0            0            1]

6. Perspective (full homography):
   H = [h00 h01 h02]
       [h10 h11 h12]
       [h20 h21 h22]

================================================================================
HOMOGRAPHY TRANSFORMATION
================================================================================

Forward Transformation:
    [x', y', w']^T = H * [x, y, 1]^T
    
    x' = h00 * x + h01 * y + h02
    y' = h10 * x + h11 * y + h12
    w' = h20 * x + h21 * y + h22
    
    x_norm = x' / w'
    y_norm = y' / w'

Inverse Transformation (for sampling):
    [x, y, 1]^T = H^-1 * [x', y', 1]^T
    
    Used in GridSample to determine source coordinates.

GridSample Application:
    grid = [x_norm, y_norm]  # Normalized to [-1, 1]
    stabilized = GridSample(input, grid)

================================================================================
OPENCV COMPATIBILITY
================================================================================

The homography matrix format is fully compatible with OpenCV:

OpenCV Functions:
- cv2.findHomography(src_pts, dst_pts) → Returns 3x3 homography
- cv2.warpPerspective(src, H, dsize) → Applies homography
- cv2.getPerspectiveTransform(src, dst) → Returns 3x3 homography

Conversion:
    # ONNX format (row-major)
    H_onnx = [[h00, h01, h02],
              [h10, h11, h12],
              [h20, h21, h22]]
    
    # OpenCV format (row-major, same)
    H_opencv = [[h00, h01, h02],
                [h10, h11, h12],
                [h20, h21, h22]]

The formats are identical! No conversion needed.

================================================================================
CONTROL LOOP WITH HOMOGRAPHY
================================================================================

The control loop now works with homography matrices:

Loop Body:
    1. Extract translation from frame homography
       tx = H[0, 2]
       ty = H[1, 2]
    
    2. Fuse with IMU data
       tx_fused = α * tx + (1-α) * imu_dx
       ty_fused = α * ty + (1-α) * imu_dy
    
    3. Update velocity and acceleration
       v_new = v_old + [tx_fused, ty_fused, dr]
       a_new = v_new - v_old
    
    4. Apply smoothing
       v_smoothed = v_new * smoothing_factor
    
    5. Create refined homography
       H_new = [[1, 0, tx_fused],
                [0, 1, ty_fused],
                [0, 0, 1]]

State Variables:
    - velocity [3] : [vx, vy, vr]
    - acceleration [3] : [ax, ay, ar]
    - homography [3,3] : Current homography matrix

================================================================================
APPLIER WITH HOMOGRAPHY
================================================================================

The applier applies the homography transformation:

Steps:
    1. Create coordinate grid [h,w,3]
       grid = [x, y, 1]  # Homogeneous coordinates
    
    2. Apply homography transformation
       [x', y', w']^T = H * [x, y, 1]^T
       
       x' = h00 * x + h01 * y + h02
       y' = h10 * x + h11 * y + h12
       w' = h20 * x + h21 * y + h22
    
    3. Normalize by w'
       x_norm = x' / w'
       y_norm = y' / w'
    
    4. Create sampling grid [h,w,2]
       grid = [x_norm, y_norm]
    
    5. Apply GridSample
       stabilized = GridSample(input, grid)

Valid Mask (DeshakeApplierV2):
    - Pixels where w' > 0 are valid
    - Pixels where w' <= 0 are invalid (outside image)
    - Used to identify regions needing inpainting
