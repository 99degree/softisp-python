================================================================================
ALGORITHM DETAILS
================================================================================

LensLCSBase
-----------
Purpose: Base LCS implementation with external coefficients

Inputs:
- applier [n,3,h,w] : Image tensor from upstream
- lcs_coeffs [h,w]  : Per-pixel correction coefficients (external)

Outputs:
- applier [n,3,h,w] : Corrected image tensor

Algorithm:
1. Receive external lcs_coeffs
2. Multiply image by coefficients (broadcasted across channels)
3. Output corrected image

Use Case: When coefficients are provided by calibration or coordinator

LensLCSV1
---------
Purpose: Adaptive LCS with coefficient resizing

Inputs:
- applier [n,3,h,w] : Image tensor from upstream
- lcs_coeffs [H,W]  : Full-resolution correction coefficients

Outputs:
- applier [n,3,h*,w*] : Corrected image tensor
- lcs_coeffs_resized [h*,w*] : Resized coefficient map
- lcs_coeffs_out [H,W] : Identity copy of original coeffs

Algorithm:
1. Create resize scales [1,1,0.5,0.5]
2. Resize lsc_coeffs to match image dimensions
3. Identity copy of original coefficients
4. Multiply image by resized coefficients
5. Output corrected image

Use Case: When coefficient resolution differs from image resolution

LensLCSV2
---------
Purpose: Radial LCS with analytic coefficient generation

Inputs:
- applier [n,3,h,w] : Image tensor from upstream
- lcs_params [4]     : [cx, cy, strength, falloff]

Outputs:
- applier [n,3,h,w] : Corrected image tensor
- lcs_coeffs [h,w]  : Generated coefficient map

Algorithm:
1. Split parameters: cx, cy, strength, falloff
2. Create coordinate grids (normalized to [-1, 1])
3. Calculate radius from center: r = sqrt(dx^2 + dy^2)
4. Normalize radius: r_norm = r / rmax
5. Calculate correction: coeff = 1 + strength * (1 - r_norm^falloff)
6. Multiply image by coefficients
7. Output corrected image

Use Case: When coefficients need to be generated from radial parameters

EXTENSIBILITY
================================================================================

To add a new LCS algorithm:

1. Create a new class in lens_lcs_v2.py or new file:

class LensLCSV3(MicroblockBase):
    name = "lens_lcs_v3"
    family = "lcs_lcs_v3"
    version = "v3"
    
    def build_algo(self, stage, prev_stages=None):
        # Your coefficient generation here
        pass
    
    def build_applier(self, stage, prev_stages=None):
        # Your applier here
        pass
    
    def build_coordinator(self, stage, prev_stages=None):
        return super().build_coordinator(stage, prev_stages)
    
    def build_test_algo(self, stage, prev_stages=None):
        return self.build_algo(stage, prev_stages)

2. Add to __init__.py exports:

from .lens_lcs_v3 import LensLCSV3

__all__ = [
    'LensLCSBase',
    'LensLCSV1',
    'LensLCSV2',
    'LensLCSV3'
]
