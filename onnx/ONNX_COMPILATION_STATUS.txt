================================================================================
EXPECTED ONNX STRUCTURE
================================================================================

ALGO ONNX (Statistics Calculation):
-----------------------------------
Input: RGB/YUV image [n,3,h,w]

Processing:
1. Calculate mean brightness (ReduceMean)
2. Create target brightness (Constant: 0.18)
3. Calculate brightness ratio (Div)
4. Calculate exposure value (Log, Div by log(2))
5. Clip exposure value (Clip)
6. Calculate gain (Exp)

Outputs:
- brightness [1] - Calculated brightness
- target [1] - Target brightness (0.18)
- exposure_value [1] - Exposure value in EV
- gain [1] - Gain factor (2^EV)

APPLIER ONNX (Exposure Application):
------------------------------------
Input: Image [n,3,h,w] + gain coefficient

Processing:
1. Apply exposure compensation (Mul: image * gain)
2. Clip to valid range (Clip: [0, 1])

Output:
- output [n,3,h,w] - Exposure-compensated image

================================================================================
ALGORITHM-SPECIFIC DIFFERENCES
================================================================================

AutoExposureSimple:
- ONNX Nodes: ~15
- Processing: Mean brightness over all channels
- Outputs: brightness, exposure_value, gain

AutoExposureStats:
- ONNX Nodes: ~25
- Processing: Per-channel RGB, weighted luminance
- Outputs: r_stats, g_stats, b_stats, weighted_stats, exposure_value, gain

AutoExposureYUV:
- ONNX Nodes: ~20
- Processing: YUV channel split, Y channel only
- Outputs: y_channel, u_channel, v_channel, y_stats, exposure_value, gain

AutoExposureHistogram:
- ONNX Nodes: ~20
- Processing: Grayscale conversion, histogram analysis
- Outputs: brightness, exposure_value, gain

AutoExposureMultiZone:
- ONNX Nodes: ~20
- Processing: Multi-zone weighted analysis
- Outputs: brightness, exposure_value, gain

TESTING COMPILED MODELS
================================================================================

Once ONNX models are compiled, you can test them:

1. Load and Run with ONNX Runtime:

   import onnxruntime as ort
   import numpy as np

   # Load models
   algo_session = ort.InferenceSession('onnx_out/test_autoexposure_simple/algo.onnx')
   applier_session = ort.InferenceSession('onnx_out/test_autoexposure_simple/applier.onnx')

   # Create test input
   test_image = np.random.rand(1, 3, 640, 480).astype(np.float32)

   # Run algorithm
   algo_outputs = algo_session.run(None, {'input': test_image})
   gain = algo_outputs[3]  # gain output

   # Run applier
   applier_outputs = applier_session.run(None, {'input': test_image, 'gain': gain})
   result = applier_outputs[0]

   print(f"Gain: {gain}")
   print(f"Result shape: {result.shape}")

2. Validate Exposure Calculation:

   # Check that gain is reasonable
   assert 0.25 <= gain[0] <= 4.0, "Gain out of expected range"
