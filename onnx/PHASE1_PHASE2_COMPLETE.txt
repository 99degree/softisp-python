================================================================================
ONNX STRUCTURE - ALGO STAGE
================================================================================

INPUTS:
- RGB/YUV image [n,3,h,w]

PROCESSING:
1. Calculate mean brightness: ReduceMean
2. Create target brightness: Constant (0.18)
3. Calculate brightness ratio: Div (target / current)
4. Calculate exposure value: Log(ratio) / log(2)
5. Clip exposure value: Clip (min_ev, max_ev)
6. Calculate gain: Exp(EV)

OUTPUTS:
- stats [1] - Calculated brightness statistics
- target [1] - Target brightness (0.18)
- brightness_ratio [1] - Target/current ratio
- exposure_value [1] - Exposure value in EV
- gain [1] - Gain factor (2^EV)

ONNX NODES:
- ReduceMean, Constant, Div, Log, Clip, Exp

================================================================================
ONNX STRUCTURE - APPLIER STAGE
================================================================================

INPUTS:
- Input image [n,3,h,w]
- Gain coefficient from algo stage

PROCESSING:
1. Apply exposure compensation: Mul (input * gain)
2. Clip to valid range: Clip (0, 1)

OUTPUTS:
- output [n,3,h,w] - Exposure-compensated image

ONNX NODES:
- Mul, Constant, Clip
